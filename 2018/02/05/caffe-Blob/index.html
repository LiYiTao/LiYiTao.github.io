caffe Blob

发表于 2018-02-05

概述

Caffe使用成为Blob的4维数组用于存储和交换数据。Blob提供了统一的存储器接口，持有一批图像或者其他数据，权值，权值更新值。
Blob在内存中表示4维数组，维度从低到高为(width_,height_,channels_,num_) 四个维度分别表示，（宽，长，通道数，图像数）。

例子：Feature Map

例如(224,224,3,2)
就表示两个张图，每张图有RGB三通道，每张图片的大小是224*224。
同样，卷积核也是这样表示的

存储

Blob是保存为一维数组的，存储方式是

  1. 行
  2. 列
  3. 通道
  4. 图像
    就是一行一行取出来拍成一维数组，取完一个通道的添加通道，取完一张图的添加图。
